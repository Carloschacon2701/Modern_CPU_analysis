# PROYECTO: Organización del Computador — Semestre 2025-3

## Plataforma moderna de análisis de ejecución CPU–Memoria–I/O

---

## Objetivo

Desarrollar una plataforma de análisis de ejecución de programas que permita observar cómo interactúan **CPU**, **memoria** y **dispositivos de entrada/salida** durante la ejecución de un proceso.

El sistema debe permitir analizar:

- Uso CPU
- Acceso memoria
- Operaciones I/O
- Comportamiento de threads

La plataforma debe simular cómo se comporta la arquitectura del computador ante diferentes cargas de trabajo, incluyendo algoritmos intensivos y cargas de datos de IA.

---

## Contexto moderno

El análisis de rendimiento es esencial en:

- Sistemas cloud
- Entrenamiento de IA
- Sistemas distribuidos
- Computación paralela

---

## Tecnologías obligatorias

Deben utilizar:

- **Antigravity IDE**
- **OpenAI Codex**
- **Python o Rust**
- **Dashboard web moderno**

### IA obligatoria

- Generación inicial de módulos
- Refactorización
- Pruebas automatizadas

> Antigravity permite usar agentes de IA que ejecutan tareas completas de programación dentro del IDE y coordinan múltiples procesos de desarrollo simultáneamente.

---

## Requerimientos funcionales obligatorios

El sistema debe implementar obligatoriamente los siguientes módulos:

### 1. Monitor de CPU

Debe mostrar:

- Utilización CPU
- Número de instrucciones ejecutadas
- Tiempo de ejecución

### 2. Monitor de memoria

Debe mostrar:

- Uso de memoria
- Accesos lectura/escritura
- Latencia simulada

### 3. Monitor de I/O

Debe mostrar operaciones de:

- Disco
- Red
- Dispositivos simulados

### 4. Visualización de procesos

Mostrar:

- Procesos activos
- Threads
- Prioridad

### 5. Simulación de carga de trabajo

El sistema debe permitir ejecutar:

- Algoritmo CPU intensivo
- Carga de datos simulada
- Procesamiento paralelo simple

---

## Innovación obligatoria

Cada grupo debe analizar el comportamiento del sistema al ejecutar un **algoritmo de procesamiento de datos o IA simple**.

Ejemplos:

- Procesamiento de matriz
- Clasificación de datos
- Simulación de entrenamiento pequeño
